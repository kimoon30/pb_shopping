# 반응형 디자인 가이드

## 📱 반응형 브레이크포인트

페이백쇼핑 랜딩페이지는 Tailwind CSS의 반응형 브레이크포인트를 사용합니다:

- **Mobile**: < 640px (기본)
- **SM (Small)**: ≥ 640px (작은 태블릿)
- **MD (Medium)**: ≥ 768px (태블릿)
- **LG (Large)**: ≥ 1024px (데스크톱)
- **XL (Extra Large)**: ≥ 1280px (큰 데스크톱)

## 🎨 적용된 반응형 기능

### 1. **Header (네비게이션)**
- ✅ 모바일: 햄버거 메뉴로 전환
- ✅ 태블릿/데스크톱: 전체 네비게이션 바 표시
- ✅ 높이: 70px (모바일) → 100px (데스크톱)

### 2. **Hero 섹션**
- ✅ 배경 이미지: 모바일에서 숨김, 데스크톱에서 표시
- ✅ 헤드라인 크기: 40px → 68px (반응형)
- ✅ 문의 폼: 모바일 가운데 정렬, 데스크톱 오른쪽 정렬
- ✅ 패딩 조정: pt-[100px] → pt-[164px]

### 3. **ContactForm (문의 폼)**
- ✅ 너비: 100% (모바일) → max-w-[568px] (데스크톱)
- ✅ 입력 필드: 1열 (모바일) → 2열 (태블릿+)
- ✅ 플랫폼 버튼: 작은 크기에서 큰 크기로 조정
- ✅ 패딩: p-5 → p-10 (반응형)

### 4. **PainPoints 섹션**
- ✅ 그리드: 1열 → 2열 (MD) → 3열 (LG)
- ✅ 아이콘 크기: 50px → 84px
- ✅ 제목 크기: 22px → 30px
- ✅ 카드 패딩: p-6 → p-[60px_30px]

### 5. **Solutions 섹션**
- ✅ 그리드: 1열 → 2열 (MD) → 3열 (LG)
- ✅ 이미지 크기: 120px → 160px
- ✅ 제목 크기: 24px → 35px
- ✅ 배지 텍스트: 12px → 16px

### 6. **Beta Promotion 섹션**
- ✅ 그리드: 1열 → 2열 (LG)
- ✅ 레이아웃: 세로 (모바일) → 가로 (SM+)
- ✅ 아이콘 크기: 40px → 58px
- ✅ 제목 크기: 20px → 28px

### 7. **FAQ 섹션**
- ✅ Q/A 아이콘 크기: 6x6 → 7x7
- ✅ 질문 텍스트: 14px → 20px
- ✅ 패딩: p-4 → p-10
- ✅ 화살표 아이콘: 5x5 → 7x7

### 8. **Testimonials (고객 후기)**
- ✅ 그리드: 1열 → 2열 (MD)
- ✅ 아바타 크기: 14x14 → 20x20
- ✅ 제목 크기: 18px → 28px
- ✅ 카드 패딩: p-6 → p-[60px_30px]

### 9. **Brands (브랜드 로고)**
- ✅ 그리드: 2열 → 3열 (SM)
- ✅ 로고 크기: 100px → 157px (max-width)
- ✅ 제목 크기: 28px → 45px

### 10. **CTA 섹션**
- ✅ 배경 이미지: 모바일 숨김
- ✅ PASS 카드: 세로 레이아웃 (모바일) → 가로 (MD)
- ✅ 앱 아이콘: 80px → 100px
- ✅ 버튼: 전체 너비 (모바일) → 자동 (SM+)
- ✅ 헤드라인: 24px → 40px

### 11. **Footer**
- ✅ 레이아웃: 세로 (모바일) → 가로 (LG)
- ✅ 텍스트 크기: 12-13px → 14px
- ✅ 정보: 세로 배치 (모바일) → 가로 배치 (SM+)

## 💡 사용된 Tailwind 반응형 클래스 예시

```html
<!-- 크기 조정 -->
<h1 class="text-[28px] sm:text-[35px] md:text-[40px] lg:text-[45px]">

<!-- 그리드 레이아웃 -->
<div class="grid grid-cols-1 md:grid-cols-2 lg:grid-cols-3 gap-4">

<!-- 패딩 조정 -->
<div class="px-4 sm:px-8 md:px-16 lg:px-[360px]">

<!-- 표시/숨김 -->
<div class="hidden lg:block">
<button class="lg:hidden">

<!-- Flexbox 방향 -->
<div class="flex flex-col md:flex-row">
```

## 🧪 테스트 방법

### 1. 개발 서버에서 테스트
```bash
npm run dev
```
브라우저 개발자 도구에서 반응형 모드로 테스트

### 2. 다양한 화면 크기 테스트
- **모바일**: 320px - 639px
- **태블릿**: 640px - 1023px
- **데스크톱**: 1024px+

### 3. 주요 테스트 포인트
- ✅ 햄버거 메뉴가 모바일에서 작동하는가?
- ✅ 문의 폼 입력 필드가 올바르게 배치되는가?
- ✅ 카드 그리드가 화면 크기에 따라 변경되는가?
- ✅ 텍스트가 모든 화면에서 읽기 쉬운가?
- ✅ 이미지가 적절한 크기로 표시되는가?

```javascript
      'md': '768px',
      'lg': '1024px',
      'xl': '1280px',
      '2xl': '1536px',
    }
  }
}
```
